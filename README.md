## Setup project

### Step 1. Project setup

Before starting work with the project, run the command:

```
yarn setup
```

This command will install the project dependencies and add an entry to `/etc/hosts`

### Step 2. Environment variables

Add file `.env` similarly to `.env.example`

### Step 3. Launch of the project

To run the project run the command

```
yarn start
```

Done! The project will start at https://local-art.spoonflower.com:3000
